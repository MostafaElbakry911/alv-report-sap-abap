## Project Name
Custom Employee Master Data ALV Report

## Transaction Code
ZMT_ALV_EMP

## Overview
This project is a custom SAP ABAP ALV report developed to display employee master data using SAP HCM infotypes.

## Technical Details
- SAP ABAP
- ALV Reporting
- Logical Database (LDB)
- SAP HCM Infotypes:
  - PA0000 (Actions)
  - PA0001 (Organizational Assignment)
  - PA0002 (Personal Data)
  - PA0008 (Basic Pay)
  - PA0185 (Personal IDs)

## Features
- Custom transaction code execution
- Dynamic selection screen
- Customized ALV layout
- Backend data processing and debugging

## Purpose
To provide a structured and efficient report for employee master data analysis in SAP HCM.
